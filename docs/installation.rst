============
Installation
============


Stable release
--------------

To install Catana, run this command in your terminal:

.. code-block:: console

    $ pip install catana

This is the preferred method to install Catana, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io/en/stable/
.. _Python installation guide: https://docs.python-guide.org/starting/installation/


From sources
------------

The sources for Catana can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/catana-research/catana

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/catana-research/catana/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/catana-research/catana
.. _tarball: https://github.com/catana-research/catana/tarball/master
